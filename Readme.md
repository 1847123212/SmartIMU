Description
========
SmartIMU 是一個集成微控制器 ( STM32F103T )、感測器 ( MPU-9250、MS5611 ) 於一體的模組，透過 UART、SPI、I2C 來讀取計算出載體的角度、加速度、速度、位移、高度 ... 等等資訊。
